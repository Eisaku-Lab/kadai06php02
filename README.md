# 体重記録アプリ - README

## ①課題名
体重記録アプリ

## ②課題内容（どんな作品か）
体重管理アプリケーション。年末年始の体重を記録し、推移を折れ線グラフで可視化。

## ③アプリのデプロイURL
http://ek-deploy.sakura.ne.jp/kadai06weight/select.php

## ⑤工夫した点・こだわった点

### 制作意図
- 年末年始に太った状況の記録
- Chart.jsライブラリを活用したビジュアライゼーションの実装

### 工夫した点
#### UI/UX面  
- Chart.jsによる折れ線グラフ
- `type="date"`で日付ピッカー実装
- `type="number" step="0.1"`で小数点入力対応

#### 設計面
- db情報を都度書くのではなくfuncs.phpに集約
